# Nightwatch 文档

这是 [nightwatchjs.org](http://nightwatchjs.org) 网站的文档资源。

Nightwatchjs 是一个UI自动化测试框架由[Node.js](http://nodejs.org/)驱动. 它使用 [Selenium WebDriver API](https://code.google.com/p/selenium/wiki/JsonWireProtocol).
